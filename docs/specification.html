IRIS Traffic Management System:

Features

  • Polished map-based interface
  • DMS sign control
  • Continuous traffic data collection / archiving
  • Ramp meter operation
  • Lane Control Signal operation
  • Camera control
  • Integrated database administration

Protocols

  • NTCIP Class B (RS-232)
  • Wavetronix SmartSensor (detection)
  • 3M Canoga (detection)
  • Vicon Switcher (camera control)
  • Mn/DOT 170 Protocol (detection, ramp metering, LCS)

Requirements

  • Server
    • Pentium IV (or similar)
    • 1 GB RAM
    • RS-232 Expansion (Digi, Cyclades or similar)
    • Red Hat / Fedora Core Operating System
    • Java 1.5 Platform
    • RXTX Serial Communication Package
    • PostgreSQL 7.4 Database
    • Apache 2 Web Server
  • Client
    • Pentium IV (or similar)
    • 512 MB RAM
    • Any Operating System
    • Java 1.5 Platform
  • Administration
    • ESRI ArcGIS (for client maps)
    • LDAP server (Novell NDS or other)
