1. Introduction

IRIS is the software used by the RTMC staff to control field devices and collect traffic data.

2. Installation

Install the Java Runtime Environment. This must be done by someone with local admin privileges. If you attempt to install it without proper privileges, the installation will fail and you will get a "bad installation" error. For installation instructions and trouble shooting help, see the Java Installation Guide.

3. Joysticks

An optional software module must be installed to control camera PTZ with a joystick. First, a couple of external packages, Python 2.4, and the pygame library are needed. This requires admin privilegdes. Also, this script must be saved in the C:\Python24\Scripts folder with the name pygame_joy.py.

4. Running

To run IRIS, click here.

IRIS requires unrestricted network and PC access. By design, Java Web Start restricts this access. You will be prompted to allow unrestricted access but advised not to do so. Disregard this advice and click the "Start" button to allow access. After that, you may start up the Java Web Start Application Manager and run it from there. The second time the program is run, you will be prompted to create a shortcut on the desktop to run IRIS directly. This is a good idea.
