Variable Speed Advisory Calculation

Station Parameters
  • Speed Limit (L)
  • Mile Point (M)
Station Data (Average of 2 most recent 30-second samples)
  • Flow (Q)
  • Speed (U)
  • Density (K)
Station Results
  • Advisory Speed (A)

For each corridor, iterate through all mainline stations using a "section" of four stations at a time. Ordinarilly, consecutive stations are used, but any station which does not have valid flow, speed and density data should be skipped. These stations are labeled S0 through S3 (from downstream to upstream).

  1. Check that M0 - M3 < 4 mi. If not, stop processing section.
  2. Check that U0 is lower than U1, U2 and U3. If not, stop processing section.
  3. Calculate speed drop at last station.
    • Ud = U1 - U0
  4. Check that Ud > U2 - U1 and Ud > U3 - U2. If not, stop processing section.
  5. Calculate speed drop threshold (D). This is normally 10 mph, but may be up to 40 mph if U1 < 40 mph.
  6. Check that Ud > D. If not, stop processing section.
  7. Calculate shock wave (Sw) speed.
    • Qu = (Q2 + Q3) / 2
    • Qd = (Q0 + Q1) / 2
    • Ku = (K2 + K3) / 2
    • Kd = (K0 + K1) / 2
    • Sw = (Qd - Qu) / (Kd - Ku)
  8. Calculate control distance (C).
    • C = Ud / 30
    • If Sw < -5, add 0.5 to C.
    • Limit C to range of [0.5, 1.5] (configurable).
  9. Find station nearest to M0 - C. This is Sc.
  10. Calculate the control range (Mr).
    • Mr = M0 - Mc
  11. Calculate the speed range (Ur).
    • Ur = Uc - U0
  12. For each station n, from S0 to Sc, set the advisory speed (An).
    • lerp = (M0 - Mn) / Mr
    • An = U0 + Ur * lerp
    • Limit An to range of [Un - 10, Ln]

Applying Speed Advisory to ILCS

For each LCS array with a deployed VSL action plan, determine whether the VSL indication should be displayed or not.

  1. Find the stations immediately upstream and downstream of the LCS array.
  2. If neither station has an advisory less than the speed limit, do not display a VSL indication.
  3. Using linear interpolation, find the speed advisory at the location of the LCS array.
  4. Quantize the speed advisory to 5 mph granularity. This is done by rounding down to the next lower 5 mph (floor).
  5. Limit advisory to no less than 5 mph below previous 30 second advisory.
  6. Limit advisory to no less than configurable minimum advisory (30 mph).
